Fundamentals of HTML

HTML is a set of tags and their attributes that define different parts of web document and inform web browsers how to display them. An HTML document is a text file that contains the information you want to publish. FSIt also contains embedded instruction called elements.These elements indicate to the Web how to present document. the document. HTML has many static elements like text and list, hyperlink, images, tables and frame. HTML also has dynamic elements like forms, multimedia, Java Applet, JavaScript, style sheet and dynamic HTML.

World Wide Web Consortium (W3C) defines specificationS for HTML. The W3C has defined HTML as an application of the Standard Generalized Markup Language (SGML). Most recent version of specification for the HTML is 4.01.
